## Versioning System

### Format
`[MovementName]-Mov[#]-rev[XX]-[StepDesc].musicxml`

### Examples
- `Fire-Mov3-rev01-ExpandedSections.musicxml` - Initial expansion and notation improvements
- `Fire-Mov3-rev02-TechnicalEdit.musicxml` - Technical/playability edits
- `Fire-Mov3-rev03-CohesionFlow.musicxml` - Cohesion and flow refinement
- `Fire-Mov3-rev04-FinalPolish.musicxml` - Final polish and melodic reassignment

## Fire-Evoking Techniques

The Fire movement uses these musical techniques to evoke fire:
- Rapid passages (16th notes) = flickering flames
- Tremolo = sustained fire intensity
- Staccato/staccatissimo = sparks and embers
- Crescendo/diminuendo = fire growing/dying
- Accelerando/ritenuto = fire intensifying/calming
- Double stops = fuller, more intense fire
- Trills = heat shimmer
- Harmonics = ethereal fire glow
- Col legno = fire crackling
- Dynamic contrasts = fire breathing
